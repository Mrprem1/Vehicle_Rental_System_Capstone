Layers:
- UI: HTML/CSS/vanilla JS under /public
- API: Express.js REST + JWT
- Tests: Playwright drives Chrome; POM encapsulates selectors and navigation
- Fixtures: JSON test data (credentials, cards)
- Reporting: HTML report (playwright-report/) + list reporter

Parallel execution: workers default (or CI=2); booking tests use offset dates per workerIndex to reduce collisions.
